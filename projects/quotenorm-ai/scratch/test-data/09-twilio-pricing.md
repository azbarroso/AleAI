# Twilio Pricing
Source: https://www.twilio.com/en-us/pricing
Captured: 2026-03-12
Format: URL (pricing page)

## Communications

### SMS
- Starts at $0.0083 per message (send or receive)

### RCS
- Starts at $0.0083 per message (send or receive)

### WhatsApp Business API
- Starts at $0.005 per message; prices vary by use case

### Conversations API
- Starts at $0.05 per active user per month

### Voice APIs
- Receiving calls: $0.0085/min
- Making calls: $0.014/min

### Email API (SendGrid)
- Free tier: 100 emails/day
- Paid plans start at $19.95/month

### Marketing Campaigns (SendGrid)
- Free trial: 100 emails/day
- Paid plans start at $15/month

### Video API
- Starts at $0.004/min

### Flex (Contact Center)
- Free allocation: 5,000 hours
- $1/active user hour OR $150/named user per month

## Authentication

### Lookup
- Starts at $0.01 per request

### Verify
- Starts at $0.05 per verification

## Conversational AI
- ConversationRelay: $0.07/min
- Transcription: $0.0350/min
- Language operators: $0.0050/min

## Builder Tools
- Studio: $0.0025 per flow execution
- TaskRouter: First 100 tasks/month free; then $0.06/task
- Elastic SIP Trunking: Origination $0.0045/min; Termination $0.007/min
- Serverless: $0.0001 per request/invocation
- Functions: 10,000 free; then $0.0001 per invocation
- Twilio for Salesforce: $15/seat/month

## Terms
- Pay-as-you-go pricing
- No long-term contracts required
- Volume discounts available
- No credit card required for free trial
