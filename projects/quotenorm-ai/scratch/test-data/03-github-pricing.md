# GitHub Pricing
Source: https://github.com/pricing
Captured: 2026-03-12
Format: URL (pricing page)

---

### Free Plan
- **Price:** $0 USD/month
- **Repositories:** Unlimited public/private
- **CI/CD Minutes:** 2,000 minutes/month (free for public repos)
- **Package Storage:** 500MB (free for public repos)
- **Features:** Dependabot security updates, Issues & Projects, community support
- **Add-ons available:** GitHub Copilot, GitHub Codespaces

### Team Plan
- **Price:** $4 USD per user/month (first 12 months promotional rate)
- **Repositories:** Unlimited public/private
- **CI/CD Minutes:** 3,000 minutes/month (free for public repos)
- **Package Storage:** 2GB (free for public repos)
- **Codespaces:** Included with flexible compute ($0.18/hr) and storage ($0.07/GB/month)
- **Key Features:** Repository rules, multiple pull request reviewers, draft PRs, code owners, required reviewers, Pages/Wikis, environment deployment branches, web-based support
- **Add-ons:** Secret Protection, Code Security

### Enterprise Plan
- **Price:** Starting at $21 USD per user/month (first 12 months promotional rate)
- **CI/CD Minutes:** 50,000 minutes/month (free for public repos)
- **Package Storage:** 50GB (free for public repos)
- **Key Features:** All Team features plus data residency, Enterprise Managed Users, SCIM provisioning, enterprise account management, environment protection rules, enterprise-wide repository rules, Audit Log API, SOC 1/2 Type 2 reports, FedRAMP ATO, SAML SSO, advanced auditing, GitHub Connect
- **Support:** Premium Support available (30-min SLA on urgent tickets, 24/7 web/phone support)

## Add-On Services
- **GitHub Copilot:** Free tier: 2,000 completions + 50 chat requests/month; paid plans available
- **GitHub Codespaces:** $0.18/hr compute, $0.07/GB/month storage
- **Premium Support:** 30-minute SLA on urgent tickets, 24/7 support
- **Git Large File Storage:** $5/month for 50GB bandwidth and 50GB storage
