## Plan Overview

Linear offers four pricing tiers: Free, Basic, Business, and Enterprise.

### Free Plan
- **Price:** $0
- **Cost Model:** Free for everyone
- **Key Limits:**
  - 2 teams maximum
  - 250 issues limit
  - 10MB file uploads
  - Unlimited members
- **Core Features:** Issues, projects, cycles, initiatives, customer requests, API/webhook access, import/export, triage, pulse, issue sync
- **Integrations:** Slack and GitHub
- **AI Features:** AI agents with agent platform and MCP access

### Basic Plan
- **Price:** $10 per user/month (when billed yearly)
- **Billing:** Annual billing option specified
- **Key Limits:**
  - 5 teams maximum
  - Unlimited issues
  - Unlimited file uploads
  - Unlimited members
- **Included:** All Free features plus admin roles

### Business Plan
- **Price:** $16 per user/month (when billed yearly)
- **Billing:** Annual billing option specified
- **Key Limits:**
  - Unlimited teams
  - Unlimited issues
  - Unlimited file uploads
  - Unlimited members
- **Additional Features:** Private teams, guest accounts, Triage Intelligence, Linear Insights, Linear Asks, Issue SLAs, Zendesk and Intercom integrations

### Enterprise Plan
- **Price:** Contact sales (custom pricing)
- **Billing:** Annual billing only
- **Features:** All Business features plus sub-initiatives, advanced Linear Asks, dashboards, SAML/SCIM, advanced security, migration and onboarding support
- **Security:** HIPAA compliance available

## Common Features Across All Plans
- Sub-teams and team owners
- Guest accounts
- Data warehouse sync
- Google SSO plus SAML (Enterprise enhanced)
- IP restrictions
- SCIM provisioning
- Domain claiming
- Audit logging
- Third-party app management
- Priority support with account manager availability on higher tiers
